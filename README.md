1.project title-Modular Furniture for Dorm Rooms.

2. Project Overview
Modular Furniture for Dorm Rooms is a simple and interactive web application designed to showcase space-saving furniture solutions for students and small living spaces. The project highlights smart designs like foldable desks, storage beds, and multi-purpose furniture.

3. Objective
Students living in dorm rooms often struggle with limited space. This project provides a solution by presenting compact, modular, and multi-functional furniture ideas in an easy-to-use interface.

4.Features-

🏠 Home page with attractive UI
🔐 Login validation system
🛍️ Product showcase (Modular furniture)
💬 Feedback form for user interaction
📱 Responsive design (basic)

5.Technologies Used

HTML5 – Structure of the web pages
CSS3 – Styling and layout
JavaScript – Functionality and interactivity

8.Usage of the Project

Navigate using the menu (Home, Products, Feedback)
Click on Explore button to open Login
Enter valid email and password
After successful login, access the Products page
View different modular furniture designs
Use the Feedback form to send your message.
